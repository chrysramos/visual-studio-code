:::CRIAR TEMA PARA VSCODE:::

1-Install node.js (if you haven't already done)
2-Install yo (if you haven't already done) by executing npm install -g yo
3-Install the Yo generator for code: npm install -g generator-code
4-Run yo code and select New Color Theme
5-Follow the instructions (define the .tmTheme file, theme name, ui theme etc.)
6-The generator creates a directory for your extension with the name of the theme in your current working directory.
